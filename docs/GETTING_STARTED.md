# Getting Started

Welcome to probable-spork! This guide will help you get up and running quickly.

## Prerequisites

- Git installed on your machine
- Basic command-line knowledge

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/lannathompson65-arch/probable-spork.git
cd probable-spork
```

### 2. Verify Installation

```bash
# Check that all files are present
ls -la
```

### 3. Next Steps

- Read the [README.md](../README.md) for an overview
- Check out the [CONTRIBUTING.md](../CONTRIBUTING.md) if you want to contribute
- Explore the project structure

## Troubleshooting

### Clone fails

Make sure you have git installed:
```bash
git --version
```

### Permission denied

If you encounter permission issues, ensure you have the proper access to the repository.
